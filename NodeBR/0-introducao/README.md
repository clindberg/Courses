# Módulo 0 - Introdução

## Iniciando um projeto Node

Criando um projeto node
'npm init - y'

Excutando o script index.js
'node index.js'

Executando o projeto 0-introdução

    - Adicionar no arquivo *package.json* dentro de script
        '"dev": "node index.js",'
    - Executar o comando no prompt
        'npm run dev'
